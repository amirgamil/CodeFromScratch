Our Journey

Swift Basics

Welcome to the world of iOS development! Here, we're going to covers all of the basics of the Swift language. We take a look at what we use to develop apps as well as both variables and constants and how we use them to store data.

Time: 30 minutes

Level: Beginner

Types and Operators

In this course, we're going to learn about the concepts of types, type safety and the importance of security (with types) in our apps. Finally, we take a look at operators and refresh our memories with a bit of basic mathematics.

Time: 45 minutes

Level: Beginner

Collections and Control Flow

Data can be stored in different ways, one of which are collections. Here, we'll learn about arrays, dictionaries, and how we can use them. After that, we'll look at how we can use control flow to run a block of code a set number of times.

Time: 2 hours

Level: Beginner

Functions

Functions are a fundamental building block of any computer that allow us to write chunks of resuable code. After a bit of practice with building our own functions, we'll look at multiple return types, customizing argument labels, and function scope

Time: 1 hour

Level: Beginner

Object Oriented Programming

Swift allows us to declare custom data types that are much more complex than the primitive data types we know about. We're going to learn about structs, classes and their differences. After that, we'll cover some trickier concepts like inheritance.

Time: 2 hours

Level: Beginner

Building Our First iPhone App

Time to put our skills to practice! Here, we'll take a look at some basic concepts like views, view controllers and auto-layout. We'll use these skills to build a travel app that shows random locations and facts with the click of a button.

Time: 2.5 hours

Level: Beginner
